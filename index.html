Mi Proceso

  1. Conversación

    Primero vemos con el cliente el concepto e identidad coorporativa que posee y discutimos las necesidades que require.

  2. Idea

    Después de obtener toda la información propongo la idea de diseño, interfaz, usabilidad y experiencia de usuario.

  3. Implementación

    Una vez que el cliente apruebe todo, paso a implementar en código todo el desarrollo web acordado con el cliente.

  4. Evaluación

    Despues de terminar con la programación, hago todas las pruebas necesarias de compatibilidad y buena práctica.

  5. Entrega

    Una vez que todo este en orden y el trabajo corriendo en internet, procedo a una entrega satisfactoria.
